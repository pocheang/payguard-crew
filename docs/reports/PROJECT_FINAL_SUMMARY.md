## 🎓 最佳实践

### 1. 部署
```bash
# 快速启动
./deploy.sh  # Linux/Mac
.\deploy.ps1 # Windows

# 或使用Make
make dev     # 开发环境
make prod    # 生产环境
```

### 2. 配置
```bash
# 复制环境配置
cp .env.production .env

# 修改关键配置
# - JWT_SECRET_KEY: 修改为随机字符串
# - API_KEYS: 设置API密钥
# - REDIS_URL: 配置Redis连接
```

### 3. 监控
```bash
# 访问Prometheus指标
curl http://localhost:8000/metrics

# 集成Grafana（可选）
# 导入dashboard模板
```
