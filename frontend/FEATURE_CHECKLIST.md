## 测试建议

### 1. 单笔审计流程测试
```
1. 进入"单笔审计"页面
2. 点击"低风险场景"填充数据
3. 点击"开始审计"
4. 查看审计结果
5. 如果决策为"review"，点击"创建人工审核"
6. 验证跳转到待审核页面
```

### 2. 批量审计流程测试
```
1. 进入"批量审计"页面
2. 点击"添加 20 笔样例"
3. 调整最大并发数为 10
4. 点击"批量审计"
5. 观察进度条
6. 查看结果汇总
7. 点击"查看详细结果"
8. 点击"导出报告"
```

### 3. 审核工作流测试
```
1. 进入"待审核"页面
2. 查看待审核列表
3. 点击"分配给我"
4. 点击"查看详情"
5. 填写审核意见
6. 点击"开始审核"
7. 选择"批准"或"拒绝"
8. 添加评论
9. 验证状态更新
```

### 4. 报告查询测试
```
1. 进入"报告查询"页面
2. 选择筛选条件
3. 点击"查询"
4. 勾选记录
5. 点击"全选"
6. 点击"导出CSV"或"导出Excel"
7. 验证文件下载
```

### 5. 仪表盘测试
```
1. 完成若干审计后，进入"仪表盘"
2. 点击"刷新数据"
3. 查看统计卡片更新
4. 查看图表显示
5. 查看最近记录列表
```

## 🔍 已知限制

1. 图表需要有数据才能显示（首次使用显示空状态）
2. 导出功能依赖后端API返回正确的blob格式
3. 部分功能（如查看审计报告详情）显示开发中提示
4. 实时数据需要手动刷新
